# 健身計時器 Fitness Timer

一個專為健身訓練設計的組間計時 PWA，可安裝到 iPhone 主畫面，支援離線使用。

## 功能

- 按一個按鈕切換訓練 / 休息，自動記錄組數
- 即時顯示當前計時、累積訓練時間、累積休息時間
- 結束後顯示完整訓練統計
- 音效提示（切換訓練 / 休息 / 結束）
- PWA：可加入 iPhone 主畫面，離線也能使用

## 使用方式

1. 用手機 Safari 開啟網址
2. 分享 → 加入主畫面
3. 開始訓練 → 每完成一組按一下 → 結束訓練

## 技術

- 純 HTML / CSS / JavaScript，無任何框架依賴
- Web Audio API（音效）
- Service Worker（離線快取）
- PWA Manifest
